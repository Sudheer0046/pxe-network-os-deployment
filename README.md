# pxe-network-os-deployment
Windows OS installation over LAN using PXE (Serva) enables network-based OS deployment without using a bootable USB or DVD. Serva acts as a PXE server to deliver Windows installation files to client machines over the network, allowing faster and centralized OS installation.


# PXE Network OS Deployment

This project demonstrates how to install Windows OS over a network using PXE (Serva) without using a bootable USB.

---

## 🎯 Objective
To deploy Windows OS to client systems over LAN using PXE boot.

---

## ⚙️ Overview
- Configured PXE server using Serva  
- Enabled network boot on client systems  
- Installed Windows OS over LAN  

---

## 🛠 Tools Used
- Serva  
- Windows ISO  
- LAN Network
